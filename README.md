# 💰 Smart Expense Tracker Pro

An intelligent expense tracking application with OCR receipt scanning, voice input, AI predictions, and voice assistance.

## Features

- 📝 **Manual Entry** - Traditional form-based input
- 🎤 **Voice Entry** - Speak to add expenses
- 📷 **OCR Scanning** - Upload receipt images
- 🤖 **Voice Assistant** - Query data via voice
- 🔮 **AI Predictions** - ML-based expense forecasting
- 📊 **Analytics** - Visual insights and reports
- 💾 **Database** - Persistent SQLite storage
